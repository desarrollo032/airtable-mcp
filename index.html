Airtable MCP Server

Bienvenido a la documentación de tu servidor MCP para Airtable.

Descripción

Este proyecto expone herramientas MCP para interactuar con Airtable mediante operaciones CRUD y gestión de bases y tablas.

Estructura

  • docs/ – Documentación del proyecto
  • src/python/ – Código fuente de MCP Server
  • .env – Variables de entorno

Herramientas MCP disponibles

  • list_bases – Lista todas las bases accesibles.
  • list_tables – Lista todas las tablas de una base.
  • list_records – Lista registros de una tabla.
  • get_record – Obtiene un registro específico.
  • create_records – Crea registros nuevos.
  • update_records – Actualiza registros existentes.
  • delete_records – Elimina registros.
  • set_base_id – Cambia la base activa.

Notas

  • Este MCP funciona solo a nivel de backend; no hay frontend por defecto.
  • Para interactuar con el servidor, se recomienda usar herramientas como curl o Postman, o desarrollar un cliente web aparte.
  • El token de Airtable se gestiona mediante variables de entorno (AIRTABLE_PERSONAL_ACCESS_TOKEN y AIRTABLE_BASE_ID).
