Términos de Servicio

Resumen: Este servicio es para desarrollo personal y uso limitado. Eres responsable del uso que hagas de la integración con Airtable.

1. Aceptación de los Términos

Al utilizar este servidor MCP para Airtable, aceptas estos términos de servicio. Si no estás de acuerdo, no utilices este servicio.

2. Descripción del Servicio

Este servidor MCP proporciona:

  • Integración con Airtable a través del protocolo MCP
  • Soporte para formatos JSON y TOON optimizados para IA
  • Autenticación OAuth 2.0 segura con Airtable
  • Herramientas para consultar y manipular datos de Airtable

3. Uso Permitido

Este servicio está diseñado para:

  • Uso personal y de desarrollo
  • Integración con asistentes de IA (como ChatGPT)
  • Automatización de tareas con Airtable
  • Prototipado y pruebas de conceptos

4. Uso Prohibido

Importante: No utilices este servicio para actividades que violen los términos de Airtable o leyes aplicables.

Queda estrictamente prohibido:

  • Violación de los términos de servicio de Airtable

  • Distribución masiva o uso público sin consentimiento
  • Actividades que comprometan la seguridad o privacidad de datos
  • Uso para spam, abuso o actividades maliciosas

5. Responsabilidades del Usuario

Como usuario, eres responsable de:

  • Mantener la confidencialidad de tus credenciales de Airtable
  • Utilizar el servicio de manera ética y legal
  • Reportar cualquier vulnerabilidad de seguridad
  • Respetar los límites de rate limiting de Airtable
  • Backup de tus datos importantes antes de usar el servicio
